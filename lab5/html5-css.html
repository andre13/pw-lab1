CSS

Cascading Style Sheets (CSS) é uma folha de estilos que permite estilizar uma página web (cores, fontes, espaçamento, etc.)

Tipos de Seletores Utilizado? Descrição da Utilização
universal done Definição global do box-sizing e questões relacionadas com a font do site
tipo done Redefinição dos elementos h3, h5, body, img, iframe, span, pre e details
classe done Definição de estilos para serem consumidos como classes tais como emoji, iframe, header, nav, main, footer e grid-container
id done Estilização do titulo/logo do site (id #main_title) e também do menu ativo (id #active_menu_tab)
atributo done Utilização do atributo target="_blank" na página de Informações, no hiperlink "https://www.london.gov.uk/"
pseudo-classe done Utilizado para definir as pseudo-classes link, visited, hover, active das ancoras "a" utilizadas em hiperlinks e no menu
pseudo-elemento done Utilizado o first-letter e first-line na página Introdução
